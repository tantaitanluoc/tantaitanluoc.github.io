### *Làm thế nào để tìm max của hai số mà không dùng đến cấu trúc rẽ nhánh?*
### *How do I find out the maximum of two number without branching?*

```js 
function max(x,y){
	return x ^ ((x ^ y) & -(x < y));
}
```
